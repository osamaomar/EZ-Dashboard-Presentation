Overview

This dashboard is designed to create an easy way to build a custom Google Analytics Dashboard on your own page. The Dashboard is built on top of the Google Analytics Easy Dashboard Javascript Library.

Note: At this time, each user who views a dashboard must have access to the Google Analytics account from which the dashboard data is generated. Currently you can not create a dashboard for users who do not have access to your Google Analytics data. Visit the Google Analytics website to sign up for a new account if you do not have one. Click the Create an account button and follow the on-screen instructions.

Step 1: Getting Started

Signing In

First, you must sign in on the top right of the dashboard. Sign in with your current google account to access your Google Analtics data. After signing in, choose the table ID of your desired analytics data.

Note: Visit the Google Analytics website to sign up for a new account if you do not have one. Click the Create an account button and follow the on-screen instructions.

Step 2: Customize Your Dashboard

Adding Charts

You can start adding a chart by clicking on the dotted box with the + signs. Where you click on the dotted box will determine where the chart will show up on the dashboard. Once you choose where you want to place the chart, the edit chart window will pop up.

Note: The dashboard only supports a maximum of 6 charts for this version.

Editting Charts

To edit any of the charts, click on the gear icon on the top right of each chart. Once clicked, the edit window will pop up for you to customize your chart.

Chart Types

Select the display format for your:

  • Timeline Chart: Select the primary metric, and optionally a comparative metric.
  • Pie Chart: Select the metric and dimension.
  • Bar Char: Select the primary metric, and optionally a comparative metric.
  • Column Chart: Select the primary metric, and optionally a comparative metric.

    • Metrics & Dimensions

    Metrics are individual elements of a dimension that can be measured as a sum or a ratio. For example, the dimension City can be associated with a metrics like Population, which would have a sum value of all the residents of the specific city.

    A dimension is an descriptive attribute or characteristic of an object that can be given different values. For example, a geographic location could have dimensions called Latitude, Longitude, or City Name. Values for the City Name dimension could be San Francisco, Berlin, or Singapore.

    Preview Chart

    When creating or editing charts on the chart menu, you will see that the charts are previewed real-time as you charge your chart / metric / dimensions. A loading notification will be shown as the charts are loading to the preview. Once loaded, the chart will be shown on the left side of the chart menu. The preview reflects how the chart will look like once you insert it on the EZ Dashboard.

    Delete Chart

    Undesired charts can be deleted from the EZ Dashboard. You are able to find the delete option on the chart menu. When you try to delete, you will be asked Are you sure you want to delete the chart? This is to make sure that you will not completely lose your chart(s) by accidentally clicking on the delete menu.

    Date Editor

    With the date editor, you are able to pick specific dates you want to gather data from. First you can either select or type from what day you want your data to start from. Then you can type or select which date you want the data to end from. After that, the data in all the charts of the EZ Dashboard will reflect the specified time period you chose.
